1. GIT - 버전관리 시스템(Version Control System), 이력(history)관리 시스템
	- 소프트웨어가 변화했을 때, 변화하기 이전의 파일과 구분되도록 변화를 관리
	- 대표적인 버전관리 시스템 ; GIT
2. git 설치 : git-scm.com/에서 다운로드 받아 설치.
	      git bash 파일을 실행해서 git 콘솔 버전 운용할 수 있다.
3. repository (저장소) : 프로젝트에 사용되는 파일들이 모여있는곳.

	* 저장소에는 로컬저장소 (내컴퓨터) / 원격저장소(web에서 공유가 가능하도록)가 존재한다.
	* GIT-HUB ; 원격저장소에서 가장 유명
           (저장소별로 저장이 가능, 변경사항 추적 가능, 협업 가능)
	   깃허브는 private한 프로젝트를 하려면 유료.
	모든 소스코드가 공겍너
